разработка логотипа

Волнообразные спиральные градиенты не только придают знаку динамики, но и создают образ “открытого энергетического портала”, который мгновенно переносит пользователя (“i”) в пункт назаначения.


Литера “i” сделана строчным курсивом для того, чтобы подчеркнуть общую динамику знака и создать ассоциирующийся с человеком образ.


Шрифтовая часть логотипа также соответствует общей динамике знака и выполнена строчными буквами (без заглавных прописных), чтобы подчеркнуть доступность и дружелюбность бренда для пользователя,ведь стремление компании или продукта быть именем нарицательным, а не собственным для пользователя (по примеру всем известных Pampers, Xerox, Jeep...) заслуживает отдельного доверия и лояльности со стороны клиента.

Авторизация пользователя

Авторизация доступна двумя способами: по SMS, и по постоянному паролю, сначала выбирается предпочитаемый режим. Если выбран режим “по SMS”, в поле номер телефона вводится по маске. После успешного ввода номера появляется кнопка “Выслать код”. Введя код из SMS происходит авторизация пользователя. Если по каким то причинам SMS не пришло на телефон, SMS можно переотправить не более 3 раз. Код генерируется всегда новый. Между отправками кода запускается таймер 30 секунд, пока таймер не вышел, код отправить нельзя.

Если выбран режим “По постоянному паролю” после номера телефона появляется поле для ввода постоянного пароля.

Если данные не прошли проверку, ошибки выводятся красным цветом над формой ввода. Если данные верны, происходит автоматический переход в защищенную часть приложения.
